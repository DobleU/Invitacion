# Auto-Animated SVG Drop Cap

A Pen created on CodePen.

Original URL: [https://codepen.io/alexandrevacassin/pen/XWvGLee](https://codepen.io/alexandrevacassin/pen/XWvGLee).

For the CodePen Drop Cap Challenge, this project automatically extracts the first letter of a paragraph, transforms it into a sleek SVG, and animates it with a stroke-drawing effect. Paired with an immersive video background, this fully customizable project is perfect for exploring creative typography.
