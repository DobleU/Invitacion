# Poppin' text

A Pen created on CodePen.

Original URL: [https://codepen.io/EricAlain/pen/eYXPbNr](https://codepen.io/EricAlain/pen/eYXPbNr).

Fun with JavaScript animations.

Meant to add a sort of "popcorn" effect to some text, this pen basically randomly "pops" the letters of some text off into randomized, animated beziers.

Some emojis for flavor!
